MOBIL'HOME

Nous sommes une application Web permettant aux particuliers d'échanger leur logement (appartement, maison, loft, etc.) pour de courtes vacances (d'une journée à quelques semaines). Ces échanges doivent être gratuits, mais la contrepartie est que chaque propriétaire définira certains services que les clients doivent fournir et certaines contraintes/limites qu'ils doivent respecter.
